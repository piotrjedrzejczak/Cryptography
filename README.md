# Cryptography Class

Python 3.8.0 or above

Example:

    Encrypt text with Ceasar's Cipher
    `python3 main.py -c -e`


Currently Supported Ciphers

    - Lab I
        - Caesar's Cipher (-c)
        - Affine Cipher (-a)
    - Lab II
        - Vigenere Cipher (-v)

Functionalities

    - Encryption (-e)
    - Decryption (-d)
    - Cryptoanalysis based on partially decrypted text (-j)
    - Bruteforce decryption (-k)
